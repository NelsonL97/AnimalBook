Animal Book by Kevin Feng and Nelson Le

This is Animalbook.

A solution to online harassment

As a way to help reduce the amount of prejudice in Facebook interactions, AnimalBook is a Google Chrome extension that helps reminds users not to judge others based on their skin color and appearances.

  • Install Animal Book
  • Git Repository & Source

Features

Clean, bug free code

Animal book is developed simply with just Javascript and HTML. This means no bugs, and no errors.

Easy & quick to install

The neat thing about Google Chrome extensions is that they're just a few clicks to install before you help create a better online community!

A new idea inspired by #HackHarassment

"An online world where everyone – regardless of their identity, orientation, background, or beliefs – is treated with respect and free from harassment."

No personal data tracked!

Animal Book does not keep track of any user data, so your privacy is protected.

Features & Examples

Replace user profile pictures with cute animals!

This allows you to keep your responses and interactions unbiased, and pleasantly reminds you to keep the online community a more inclusive space.
